Mamaison237 - Application de Gestion des Tâches Familiales

Mamaison237 est un prototype d'application web conçue pour simplifier l'organisation des tâches ménagères au sein des familles camerounaises, offrant une prévisualisation fonctionnelle de l'interface où les parents pourront assigner des tâches et les enfants suivre leurs responsabilités - bien que Firebase ne soit pas encore intégré (prévu dans la phase de développement avancée), le prototype démontre déjà le flux utilisateur complet avec des données simulées, une interface colorée construite avec React et des animations Framer Motion pour valider l'expérience utilisateur avant l'intégration des fonctionnalités back-end
